
# PIcamera_fadecandy
 - Stream video to multiple screens made out of LED streeps
 - Credit: Maurin Donneaud maurin@datapaulette.org

### Hardware
 - [Raspbeary-PI 3](https://www.raspberrypi.org)
 - [Fadecandy](https://github.com/scanlime/fadecandy)
